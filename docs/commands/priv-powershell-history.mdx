# priv-powershell-history

## Usage

```text
[BEBOP ~]$ priv-powershell-history
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `priv-powershell-history.x64.o`, accepts no arguments, and queues `TaskBOF`.

## Output

The BOF prints PSReadLine history location or related status.

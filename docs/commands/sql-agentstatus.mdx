# sql-agentstatus

## Usage

```text
[BEBOP ~]$ sql-agentstatus <server> [database] [linkedserver] [impersonate]
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `sql-agentstatus.x64.o` and packs up to four string arguments.

## Output

The BOF prints SQL Agent status and job data visible to the current context.

# priv-credential-manager

## Usage

```text
[BEBOP ~]$ priv-credential-manager
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `priv-credential-manager.x64.o`, accepts no arguments, queues `TaskBOF`, and the beacon executes it with the BOF loader.

## Output

Credential Manager output depends on the current user context and what the BOF can read.

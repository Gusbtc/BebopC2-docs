# priv-unquoted-service-path

## Usage

```text
[BEBOP ~]$ priv-unquoted-service-path
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `priv-unquoted-service-path.x64.o`, accepts no arguments, and queues `TaskBOF`.

## Output

The BOF prints unquoted service path findings when present.

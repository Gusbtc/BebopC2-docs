# sql-1434udp

## Usage

```text
[BEBOP ~]$ sql-1434udp <server-ip>
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `sql-1434udp.x64.o`, packs one string argument, queues `TaskBOF`, and returns BOF output.

## Output

Output depends on UDP 1434 reachability from the beacon.

# sql-checkrpc

## Usage

```text
[BEBOP ~]$ sql-checkrpc <server> [database] [linkedserver] [impersonate]
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `sql-checkrpc.x64.o` and packs up to four string arguments.

## Output

The BOF prints RPC option state for linked servers when available.

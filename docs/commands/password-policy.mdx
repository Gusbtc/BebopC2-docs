# password-policy

## Usage

```text
[BEBOP ~]$ password-policy [server]
```

## How it works

This is a Windows x64 built-in BOF. The teamserver resolves `password-policy.x64.o`, packs one optional wide-string server argument, and queues `TaskBOF`.

## Output

The BOF returns policy fields when the target server/domain is reachable from the beacon context.

# priv-token-privileges

## Usage

```text
[BEBOP ~]$ priv-token-privileges
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `priv-token-privileges.x64.o`, accepts no arguments, and queues `TaskBOF`.

## Output

The BOF prints the current token's privilege state.

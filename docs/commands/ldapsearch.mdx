# ldapsearch

## Usage

```text
[BEBOP ~]$ ldapsearch <filter> [attributes] [--count N] [--scope N] [--hostname dc] [--dn baseDN] [--ldaps]
```

## How it works

This is a Windows x64 built-in BOF. The teamserver resolves `ldapsearch.x64.o` and packs: filter, attributes, result limit, scope, hostname, base DN, and LDAPS flag. The BOF performs the LDAP work from the beacon process context and returns its own text output.

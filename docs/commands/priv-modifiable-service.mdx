# priv-modifiable-service

## Usage

```text
[BEBOP ~]$ priv-modifiable-service
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `priv-modifiable-service.x64.o`, accepts no arguments, and queues `TaskBOF`.

## Output

The BOF prints service permission findings when present.

# sql-users

## Usage

```text
[BEBOP ~]$ sql-users <server> [database] [linkedserver] [impersonate]
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `sql-users.x64.o` and packs up to four string arguments.

## Output

The BOF prints database users visible to the supplied context.

# sql-tables

## Usage

```text
[BEBOP ~]$ sql-tables <server> [database] [linkedserver] [impersonate]
```

## How it works

Windows x64 built-in BOF. The teamserver resolves `sql-tables.x64.o` and packs up to four string arguments.

## Output

The BOF prints table names from the selected SQL context.
